# Information Gathering Recon

Universal recon pack — runs the deterministic probes that mine every URL the target voluntarily exposes (FTP/static dirs, well-known files, asset directories, i18n bundles, canonical hidden paths) and pivots via cert-transparency for subdomains.

Pairs with the probe_intel_harvest content miner — output feeds credstuffing, JWT impersonation, subdomain-takeover, and bucket-listing probes.
